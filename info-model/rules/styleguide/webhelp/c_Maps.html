Maps

Well-structured maps help users find information quickly and form the basis for all navigation and linking. At Company Name, maps may be used to define an entire deliverable, as well as subparts for the deliverable. Use maps to:
  • Organize content into chapters or appendices
  • Organize related topics into single reusable units
  • Organize topics that all relate to a single user goal
  • Divide large sets of related information into smaller, more manageable units
  • Create a master glossary
  • Establish relationships between parent/child and sibling topics
Temporary maps may also be built to send a set of topics to an editor or reviewer even if these topics will not be referenced together in the final deliverable.
